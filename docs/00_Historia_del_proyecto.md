# EMA - Resumen del Proyecto

## Descripción general

EMA (Educational Material Assistant) es un asistente virtual desarrollado como un sistema web para apoyar el proceso de aprendizaje de estudiantes de nivel medio superior. Su propósito principal es recomendar material de estudio personalizado con base en el estilo de aprendizaje de cada estudiante, buscando facilitar la comprensión de los temas y mejorar la experiencia de estudio.

A diferencia de una plataforma tradicional donde todos los estudiantes reciben exactamente los mismos recursos, EMA busca adaptar las recomendaciones considerando la forma en que cada usuario aprende con mayor facilidad.

---

## Problema que resuelve

Los estudiantes suelen enfrentarse a una gran cantidad de recursos educativos disponibles en Internet sin contar con un criterio que les permita identificar cuáles son los más adecuados para ellos. Esto provoca que el tiempo de búsqueda aumente y que, en muchos casos, el material seleccionado no se adapte a sus necesidades o preferencias de aprendizaje.

EMA aborda este problema mediante un sistema que organiza materiales educativos y recomienda aquellos que mejor se ajustan al perfil del estudiante.

---

## Objetivo general

Desarrollar un asistente virtual capaz de recomendar materiales de estudio personalizados considerando el estilo de aprendizaje del estudiante, con el fin de apoyar su proceso de aprendizaje y facilitar el acceso a recursos educativos relevantes.

---

## Público objetivo

El sistema fue diseñado principalmente para estudiantes de nivel medio superior, particularmente alumnos del Instituto Politécnico Nacional (IPN), aunque su arquitectura permite adaptarse a otros niveles educativos con las modificaciones correspondientes.

---

## Funcionalidades principales

- Registro e inicio de sesión de usuarios.
- Administración de perfiles de estudiante.
- Identificación del estilo de aprendizaje del usuario.
- Recomendación de materiales de estudio personalizados.
- Gestión de materiales educativos.
- Panel de administración.
- Chat integrado para comunicación dentro de la plataforma.
- Gestión de información del usuario.

---

## Estado del proyecto

Este repositorio corresponde a la versión **2.1.4** de EMA. El desarrollo principal se realizó entre 2022 y 2023 y actualmente el proyecto se encuentra en proceso de recuperación, documentación y actualización tecnológica.

El objetivo de esta nueva etapa es reconstruir la documentación técnica, comprender nuevamente la arquitectura del sistema y facilitar su mantenimiento o evolución futura.

---

## Alcance

EMA fue concebido como una aplicación web full-stack desarrollada con Next.js y TypeScript. El sistema integra autenticación de usuarios, administración de información, almacenamiento en base de datos y recomendación de materiales educativos mediante una interfaz web orientada a estudiantes.

La documentación contenida en la carpeta `docs` describe la arquitectura, los módulos, la base de datos, la API y los principales componentes del sistema.
